# 大脚板丛のPlog编辑器

这是一个可以在手机浏览器使用的单页工具：
- 像素风 UI（标题与控件全部改为像素感风格）
- 仅保留 `黑底白字` 风格
- 支持 HEIC/HEIF 主图上传（自动转换后预览）
- 支持导入第二张图片并在画布中拖动/缩放
- 文字每行一个独立模块（回车即新模块）
- 选中模块显示白色边框（与模块有间距）+ 右上角缩放控制点
- `擦掉选中模块` 按钮可删除当前选中的文字或叠加图片
- `Live` 开关（当前为状态标记；浏览器版暂不直接导出 Live 动图）
- 黑底白字默认字体：`默认字体`（`SourceHanSansCN-Light.otf`，已内嵌并固定）
- 新增风格：`白底黑字`（排版一致，仅底色/字色反转）
- 字体导入支持长期保存：导入过的字体会保留在下拉列表中，可反复选择
- 主按钮顺序与颜色：粉红 `下载图片`、绿色 `上一步`、鹅黄 `下一步`、天蓝 `导入图片`
- 选中框控制点：左上删除、右上缩放、右下旋转+缩放

## 使用方法（小白版）
1. 打开 `index.html`。
2. 点击“导入主图”。
3. 在正文中输入多行文字（每行一个黑底白字模块）。
4. 如需贴图，点击“导入另一张图片”。
5. 在右侧画布中选中模块后直接拖动；拖右上角白色点缩放。
6. 点击“下载图片”。

## 手机上用
把 `index.html` 发到手机后，用浏览器打开即可使用。
